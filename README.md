## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation
Una vez clonado el repositorio entrar a la carpeta services y encontraras 2 carpetas adicionales en cada una de las carpetas tanto como api api-gateway-service y company-service
realizar los siguientes comando en la consola.
Tener en cuenta estar en la raiz de cada una de las carpeta api-gateway-service y company-service
```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
